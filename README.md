# Study Planner App

A simple and structured study planning application built with **Flutter**.  
The app helps students organize study tasks, track progress, and maintain consistency using streaks, XP points, and a calendar-based system.

## Features

### Profile Setup
- User profile with name, gender, and age
- Default avatar selection
- All data stored locally

### Study Task Planner
- Create daily study tasks
- Mark tasks as completed
- Organize tasks under **Stacks (subjects)**

### Calendar Planning
- Calendar view for all dates
- Plan tasks for future dates
- View tasks completed on past dates

### Streak System
- Completing tasks daily builds a streak
- Encourages consistent study habits

### XP System
- Earn XP for completing tasks
- Bonus XP for completing all tasks in a day

### Contribution Calendar
- GitHub-style calendar showing study activity
- Green squares represent productive days

### Daily Diary
- Write reflections or notes for each day
- View diary entries from previous dates via the calendar

### Extra Trackers
- Hydration tracker
- Sleep tracker
